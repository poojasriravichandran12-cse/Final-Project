# Node.js Bookstore Project

**Student Project – 30 Points**

This project implements 14 tasks using **Node.js + Axios**.

### General Users
1. Get book list – 2 pts  
2. Get book by ISBN – 2 pts  
3. Get all books by Author – 2 pts  
4. Get all books by Title – 2 pts  
5. Get Book Review – 2 pts  
6. Register new user – 3 pts  
7. Login as registered user – 3 pts  

### Registered Users
8. Add/Modify a book review – 2 pts  
9. Delete book review – 2 pts  

### Node.js Program (4 methods)
10. Get all books (Async callback) – 2 pts  
11. Search by ISBN (Promise) – 2 pts  
12. Search by Author (Async/Await) – 2 pts  
13. Search by Title (Async/Await) – 2 pts  
14. GitHub Submission – 2 pts
